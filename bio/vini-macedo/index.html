Vini Macedo

  • Project Salaf [Arabic: سلف , ancestor] (2020)
  • Github vinicyusmacedo

Vini Macedo is an infrastructure consultant at Thoughtworks, based in São Paulo, Brazil. Vini has always loved hacking and investigating how technology can be used in a creative and exploratory way.

As a part of the Salaf [Arabic: سلف , ancestor] (2020) project with Thoughtworks Arts, Vini contributed by building the cloud infrastructure for the project, and by setting up Kubeflow clusters for training machine learning models.
